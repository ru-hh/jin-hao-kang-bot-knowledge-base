# 津好康 Bot 你知 — 問答知識庫

> 如玉請假期間，同事可以用這個 repo 快速回答客戶關於津好康的任何問題。

## 快速開始（3 步驟）

```bash
# 1. 複製 repo
git clone <repo-url>

# 2. 進入資料夾
cd 津貼津好康push

# 3. 啟動 Claude Code，直接問問題
claude
```

然後直接用中文問，例如：
- 「津好康目前開發到什麼階段？」
- 「紅隊攻擊測試結果怎麼樣？」
- 「慈濟合作的成效數字？」
- 「為什麼不做多輪對話？」

Claude 會根據 repo 裡的文件回答，不會亂猜。
